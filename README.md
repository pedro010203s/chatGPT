# Sistema Muscular 3D - Masculino e Feminino

Visualização interativa 3D do sistema muscular humano com modelos masculino e feminino.

## 🚀 Características

### Modelos 3D
- **Modelo Masculino**: Sistema muscular com proporções masculinas típicas
- **Modelo Feminino**: Sistema muscular com proporções femininas típicas
- Alternância instantânea entre os modelos

### Grupos Musculares Incluídos
- **Peitoral** (Pectoralis)
- **Deltoides** (Ombros)
- **Bíceps** e **Tríceps**
- **Abdominais** (Reto Abdominal e Oblíquos)
- **Dorsais** (Latissimus Dorsi)
- **Quadríceps**
- **Isquiotibiais**
- **Panturrilhas**
- **Glúteos**

### Funcionalidades Interativas

#### Controles de Visualização
- **Vista Frontal**: Visualização frontal do corpo
- **Vista Posterior**: Visualização das costas
- **Vista Lateral**: Visualização lateral

#### Animação
- **Rotação Automática**: Rotação contínua do modelo
- **Controle de Velocidade**: Ajuste a velocidade da rotação
- **Controle Manual**: Arraste com o mouse para rotacionar
- **Zoom**: Use a roda do mouse para aproximar/afastar

#### Destacamento de Músculos
- Clique em qualquer grupo muscular para destacá-lo
- Animação de pulsação no músculo selecionado
- Painel informativo com:
  - Nome científico do músculo
  - Descrição detalhada
  - Funções principais

#### Opções de Exibição
- **Etiquetas**: Mostrar/ocultar etiquetas dos músculos
- **Modo Wireframe**: Visualizar a estrutura geométrica dos modelos

## 🎮 Como Usar

1. **Abra o arquivo `index.html`** em um navegador web moderno
2. **Selecione o gênero**: Clique em "Masculino" ou "Feminino"
3. **Explore os músculos**: Clique nos botões de grupos musculares para destacá-los
4. **Interaja com o modelo**:
   - Arraste com o mouse para rotacionar
   - Use a roda do mouse para zoom
   - Use os botões de vista para ângulos predefinidos
5. **Leia as informações**: Quando um músculo é destacado, um painel aparece com informações detalhadas

## 🔄 Diferenças entre Modelos

### Masculino
- Ombros mais largos
- Peitorais mais desenvolvidos
- Braços mais volumosos
- Cintura mais larga
- Menor definição nos glúteos

### Feminino
- Ombros mais estreitos
- Peitorais menores
- Braços mais delgados
- Cintura mais fina
- Glúteos mais pronunciados
- Proporções geralmente mais compactas

## 🚀 Executar Localmente

Simplesmente abra o arquivo `index.html` em seu navegador:

```bash
# Usando Python (se disponível)
python -m http.server 8000

# Ou usando Node.js com http-server
npx http-server

# Ou simplesmente abra o arquivo
open index.html  # macOS
start index.html # Windows
xdg-open index.html # Linux
```

Depois acesse: `http://localhost:8000`
