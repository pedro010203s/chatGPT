# Sistema Muscular 3D - Masculino e Feminino

Uma animação 3D interativa do sistema muscular humano, mostrando as diferenças entre os corpos masculino e feminino.

## Características

- **Modelos 3D Realistas**: Representação visual dos principais grupos musculares
- **Comparação Masculino/Feminino**: Diferenciações anatômicas entre os gêneros
- **Animações Interativas**: Rotação automática e controles manuais
- **Interface Intuitiva**: Controles fáceis de usar com design moderno
- **Responsivo**: Funciona em desktop e dispositivos móveis

## Músculos Incluídos

### Principais Grupos Musculares:
- **Peitorais**: Diferenças de tamanho entre masculino e feminino
- **Deltoides**: Ombros e braços
- **Bíceps e Tríceps**: Braços superiores
- **Abdominais e Oblíquos**: Core e cintura
- **Quadríceps**: Músculos da coxa
- **Glúteos**: Diferenças de proeminência
- **Isquiotibiais**: Parte posterior da coxa
- **Panturrilhas**: Músculos da perna

## Como Usar

1. Abra o arquivo `index.html` em um navegador moderno
2. Use os botões para alternar entre modelo masculino e feminino
3. Controle a animação com Play/Pause/Reset
4. Ajuste a velocidade de rotação com o slider
5. Clique nos músculos na lista para destacá-los
6. Use o mouse para rotacionar, fazer zoom e navegar na cena 3D

## Tecnologias Utilizadas

- **Three.js**: Biblioteca 3D para JavaScript
- **HTML5/CSS3**: Interface e estilização
- **JavaScript ES6**: Lógica da aplicação
- **WebGL**: Renderização 3D acelerada por hardware

## Funcionalidades

- ✅ Modelos 3D dos músculos principais
- ✅ Diferenças anatômicas entre masculino e feminino
- ✅ Animações de rotação suaves
- ✅ Controles de câmera (zoom, rotação, pan)
- ✅ Interface responsiva
- ✅ Destaque interativo dos músculos
- ✅ Controles de velocidade de animação

## Instalação

Não requer instalação. Simplesmente abra o arquivo `index.html` em um navegador que suporte WebGL.

## Requisitos

- Navegador moderno com suporte a WebGL
- JavaScript habilitado
- Conexão com internet (para carregar Three.js via CDN)
