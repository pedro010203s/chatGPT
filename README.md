# Sistema Muscular 3D - Masculino e Feminino

Uma aplicação web interativa para visualização educativa do sistema muscular humano em 3D.

## Características

- **Modelos 3D detalhados**: Visualizações separadas para anatomia masculina e feminina
- **Interatividade completa**: Rotação, zoom e controles de câmera
- **Toggle dinâmico**: Alterne entre modelos masculino e feminino instantaneamente
- **Informações educativas**: Clique nos músculos para obter informações detalhadas
- **Interface responsiva**: Funciona em desktop e dispositivos móveis
- **Animações suaves**: Transições fluidas e efeitos visuais

## Tecnologias Utilizadas

- **Three.js**: Biblioteca JavaScript para gráficos 3D
- **OrbitControls**: Controles de câmera interativos
- **HTML5/CSS3**: Interface moderna e responsiva
- **Raycasting**: Detecção precisa de cliques nos músculos

## Como Usar

1. Abra o arquivo `index.html` em um navegador web
2. Use o mouse para rotacionar e zoomar o modelo 3D
3. Clique no botão "Alternar" para trocar entre modelos masculino e feminino
4. Clique nos músculos para ver informações detalhadas
5. Use o botão "Resetar Visualização" para voltar à posição inicial

## Recursos Educacionais

- **Sistema Muscular Completo**: Visualização de todos os grupos musculares principais
- **Diferenças Anatômicas**: Comparação clara entre anatomia masculina e feminina
- **Funções Musculares**: Explicações sobre o papel de cada grupo muscular
- **Exercícios Recomendados**: Sugestões de exercícios para cada grupo muscular

## Controles

- **Rotacionar**: Clique e arraste com o mouse
- **Zoom**: Use a roda do mouse
- **Pan**: Pressione Shift + clique e arraste
- **Selecionar Músculo**: Clique diretamente no modelo 3D
- **Resetar**: Botão dedicado para voltar à visualização inicial
