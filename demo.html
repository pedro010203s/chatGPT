🏋️ Sistema Muscular 3D

Explore as diferenças anatômicas entre os sistemas musculares masculino e feminino

🎯 Características Principais

🔬 Modelos 3D Realistas

Representação visual detalhada dos principais grupos musculares com geometrias 3D precisas.

👥 Comparação Anatômica

Diferenciações claras entre as características musculares masculinas e femininas.

🎬 Animações Interativas

Rotação automática suave e controles manuais para exploração completa.

📱 Interface Responsiva

Design moderno que funciona perfeitamente em desktop e dispositivos móveis.

🎮 Como Usar

  1. Seleção de Modelo: Use os botões "Masculino" e "Feminino" para alternar entre os modelos
  2. Controles de Animação: Play para iniciar rotação, Pause para parar, Reset para voltar à posição inicial
  3. Velocidade: Ajuste o slider para controlar a velocidade de rotação
  4. Navegação 3D: Use o mouse para rotacionar, fazer zoom e navegar na cena
  5. Destaque de Músculos: Clique nos músculos na lista para destacá-los no modelo 3D

🛠️ Tecnologias

Esta aplicação foi desenvolvida usando tecnologias web modernas:

  • Three.js: Biblioteca 3D para JavaScript
  • WebGL: Renderização 3D acelerada por hardware
  • HTML5/CSS3: Interface e estilização responsiva
  • JavaScript ES6: Lógica da aplicação com classes modernas
